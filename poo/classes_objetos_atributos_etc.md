# Guia para iniciantes
## Classe
* formato a ser seguido para fazer objetos do mesmo tipo ou com características semelhantes

| NomeClasse                         |
| ---------------------------------- |
| + características (atributos)      |
| + coisas que posso fazer (métodos) |

## Instância
* momento em que um objeto é criado a partir de uma classe

## Objeto
* um objeto é a instância de uma classe
* coisa material ou abstrata que é feita a partir de um modelo e pode ser descrito por meio das suas características, comportamento e estado atual

    ### Estado
    * conjunto de todos os valores dos atributos de um objeto no momento atual
